# Install X Reply Janitor

X Reply Janitor is distributed as a downloadable browser extension zip on GitHub Releases.

Latest download:

https://github.com/bahuang081-svg/x-reply-janitor/releases/latest/download/x-reply-janitor.zip

## Chrome / Edge

1. Download the zip.
2. Unzip it.
3. Open `chrome://extensions` or `edge://extensions`.
4. Turn on Developer mode.
5. Click “Load unpacked”.
6. Select the unzipped folder.

## Firefox

1. Download and unzip the latest release.
2. Open `about:debugging#/runtime/this-firefox`.
3. Click “Load Temporary Add-on”.
4. Select `manifest.json`.

## What It Does After Install

- It runs only on `x.com` and `twitter.com`.
- It scans visible X/Twitter replies.
- It folds, hides, or dims likely bot spam.
- It stores settings locally in browser extension storage.

It does not request your X login, API token, cookies, or personal data.
