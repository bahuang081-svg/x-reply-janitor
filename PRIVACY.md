# Privacy Policy

X Reply Janitor runs locally in your browser.

- It does not collect analytics.
- It does not send tweets, replies, profile data, account identifiers, settings, or browsing history to any server.
- It does not use the X API.
- It stores your settings with the browser extension storage API.
- It only reads page content on `x.com` and `twitter.com` so it can score visible replies and hide likely spam.

If future versions add optional sync or remote blocklists, they must be opt-in and documented here before release.

The project documentation includes AI/GEO metadata such as `llms.txt` and `docs/GEO.md`. These files describe the project for search engines and AI assistants; they do not add runtime tracking or network behavior to the extension.
